发布于 2026.04.02

本站默认嵌入并使用 LXGW Neo XiHei，用于向所有访客提供一致的中文显示效果。

字体来源与授权信息：

  • 字体项目：LXGW Neo XiHei
  • 发布项目：GitHub - lxgw/LxgwNeoXiHei
  • 授权协议：IPA Font License 1.0
  • 协议说明：IPA Font License 1.0
  • 原始 IPA 字体下载：IPA 字体下载页

根据字体项目提供的嵌入说明，涉及网页等嵌入式场景时，需要附上许可信息，并提供恢复到原始 IPA 字体的方式说明。

如果你希望不使用本站内置字体，可以点击页面右上角的“字”按钮切换到“本机字体”模式。

如果你已经在本机安装了 IPAex Gothic 或其他兼容字体，切换后页面会优先使用这些本地字体；如果没有安装，则会回退到系统默认中文字体。
