十三、这一节最容易混淆的点

1. Cubemap 的采样坐标不是 UV

sampler2D 用 vec2，samplerCube 用 vec3。

这个 vec3 表示方向，不表示世界空间里的某个点。

2. 六面图顺序不能乱

顺序应当是：

对应：

3. Cubemap 通常不要垂直翻转

普通 2D 纹理常常需要 stbi_set_flip_vertically_on_load(true)。

cubemap 本身依赖六面图方向一致，通常使用 false。

4. 天空盒要去掉 view 平移

天空盒要保留旋转、去掉平移：

否则移动相机时会感觉自己在靠近天空盒边界。

5. GL_TEXTURE_WRAP_R 不能忘

Cubemap 是三维方向采样，除了 S/T，还要设置 R。

6. samplerCube 绑定的不是 GL_TEXTURE_2D

采样器类型和绑定目标要匹配：

7. 环境反射只是近似

它反射的是 cubemap 环境，不会真实反射当前场景里的动态物体。

十七、总结

立方体贴图把纹理采样从：

它的核心价值是让 Shader 能根据方向获取环境颜色。
